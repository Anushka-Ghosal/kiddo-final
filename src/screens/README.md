# Kiddo Assignment – Server Driven UI

## Overview

This project is a Server Driven UI application built using React Native, Expo, and TypeScript.

The UI is rendered dynamically from JSON instead of hardcoding screens.

## Features

- Server Driven UI
- Component Registry
- Banner Hero
- Product Grid (2×2)
- Dynamic Collection
- Campaign Switching
- Zustand Cart
- Theme Context
- FlashList
- React.memo Optimization

## Tech Stack

- React Native
- Expo
- TypeScript
- Zustand
- FlashList

## Folder Structure

```
src/
 ├── actions/
 ├── components/
 ├── data/
 ├── registry/
 ├── screens/
 ├── store/
 └── theme/
```

## Installation

```bash
npm install
```

## Run

```bash
npx expo start
```
